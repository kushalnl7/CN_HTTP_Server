HTTP SERVER Program:

1] To run the server file: python3 https.py 12000(Any Port number can be given by user, we are assuming it to be 12000)
2] To run the automated testing file: python3 testing.py 12000
3] To view png image file on browser, go to http://127.0.0.1:12000/pic.png
4] To view jpg image file on browser, go to http://127.0.0.1:12000/pic3.jpg
5] To view jpeg image file on browser, go to http://127.0.0.1:12000/pic4.jpeg
6] To view mp3 song file on browser, go to http://127.0.0.1:12000/song.mp3
7] To view mp4 video file on browser, go to http://127.0.0.1:12000/video.mp4
8] Type 'quit' on server if you want to end the server program.
